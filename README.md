# Slogistics

A high-performance logistics application built with Next.js 16 and high-security cryptography.

## Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Package Manager**: [pnpm](https://pnpm.io/)
- **Cryptography**: [@noble/curves](https://github.com/paulmillr/noble-curves) & [@noble/hashes](https://github.com/paulmillr/noble-hashes)
- **Standardization**: [Model Context Protocol (MCP)](https://modelcontextprotocol.io)

## Getting Started

First, install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
